To prevent pacman -Syu from immediately upgrading it back to the broken version, you need to edit your configuration file.

Open the config: sudo nano /etc/pacman.conf

Find the [options] section.

Uncomment or add the IgnorePkg line:

IgnorePkg = tzdata     ----> or any package other than tzdata



To go back to a previous version of a package

ls /var/cache/pacman/pkg/ | grep package_name

sudo pacman -U /var/cache/pacman/pkg/package_name-old_version-x86_64.pkg.tar.zst
