Download the latest .run installer from the PIA Linux download page to your Downloads directory

Execute the shell script:

cd ~/Downloads
sh pia-linux-*.run

login to pia with credentials and then the waybar pia module will work

Or do it via the aur package:

paru -Syu piavpn-bin
sudo systemctl enable --now piavpn.service
sudo systemctl start piavpn.service
sudo systemctl status piavpn.service
